Iteration #2
============

In this iteration we set up a basic TaskStore and TaskAPI to simulate server-side Ajax request via mock JSON data.

Just run :
```shell
npm install
npm run devserver
```

The application will be available to [http://localhost:8080](http://localhost:8000)
